Clickable Sidebar Menu: A customizable plugin for creating dynamic, interactive side menus with multi-level support for your WordPress site.


== Description ==

Clickable Sidebar Menu allows you to create customizable sidebar menus for your WordPress site. It supports multi-level submenus, color customization, and more.

= Features =
* Create sidebar menus with multiple levels of submenus.
* Customize menu and submenu colors directly from the settings page.
* Responsive and accessible.
* Easy to use shortcode for embedding the sidebar menu.

= Shortcode =
Use the shortcode `[csms_sidebar_menu]` to display the sidebar menu on your site.

== Installation ==

1. Upload the `wp-sidebar-menu` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Settings > Sidebar Menu to configure your menu and color options.
4. Use the `[csms_sidebar_menu]` shortcode to display the menu.

== Frequently Asked Questions ==

= How do I display the sidebar menu? =
Use the shortcode `[csms_sidebar_menu]` in your posts, pages, or widgets to display the sidebar menu.

= How do I customize the menu colors? =
Go to the plugin settings page under Settings > Sidebar Menu to customize the colors.

= Does this plugin support multiple menus? =
Yes, you can select which menu to display from the plugin settings page.
